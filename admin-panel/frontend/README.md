# Admin Panel Frontend

React-based admin panel for managing Vidyarthi Mobile App content.

## Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Features (To be implemented)

- Manage Books/Products
- Manage Schools
- Manage Categories
- Manage Orders
- Dashboard with analytics

## Tech Stack

- React 19
- Vite
- React Router DOM
- Axios
