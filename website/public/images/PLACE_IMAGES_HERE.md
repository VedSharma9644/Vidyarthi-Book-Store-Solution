# Place Your Images Here

## Logo Images
Place your logo files in `logo/` folder:
- `logo.png` - Main logo (recommended: 200x60px or similar aspect ratio)
- `logo-small.png` - Small logo for mobile (recommended: 100x30px)
- `favicon.png` - Favicon (recommended: 32x32px or 64x64px)

## Product Images
Place product placeholder images in `products/` folder:
- `textbook-placeholder.png` - Default image for textbooks
- `notebook-placeholder.png` - Default image for notebooks
- `stationary-placeholder.png` - Default image for stationary items
- `uniform-placeholder.png` - Default image for uniforms
- `product-placeholder.png` - Generic product placeholder

## Category Images
Place category icons in `categories/` folder:
- `mandatory-icon.png` - Icon for mandatory items
- `optional-icon.png` - Icon for optional items
- `stationary-icon.png` - Icon for stationary category

## Image Guidelines
- **Format**: Use PNG for logos/icons, JPG for photos
- **Size**: Optimize images for web (keep file sizes small)
- **Dimensions**: 
  - Logo: 200x60px or similar
  - Product placeholders: 400x600px or similar
  - Icons: 64x64px or 128x128px

## After Adding Images
Once you add images, they will be automatically available at:
- `/images/logo/logo.png`
- `/images/products/textbook-placeholder.png`
- etc.

The application will use these as fallbacks when product images from the API are not available.
